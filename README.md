# Face Detection using Python

This project detects human faces in real time using Python and OpenCV.

## 📌 Project Description
The system uses a webcam to capture video and detects human faces using the Haar Cascade Classifier.

## 🛠 Technologies Used
- Python
- OpenCV
- Haar Cascade Classifier

## 📂 Project Structure

face-detection-python
│
├── face_detection.py
├── haarcascade_frontalface_default.xml
└── README.md

## ▶️ How to Run the Project

1. Install Python
2. Install OpenCV

pip install opencv-python

3. Run the program

python face_detection.py

## 🎯 Output
The webcam opens and detects faces by drawing rectangles around them.
